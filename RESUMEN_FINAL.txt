┌──────────────────────────────────────────────────────────────────────────┐
│                        🚀 CÓMO USAR AHORA MISMO                          │
└──────────────────────────────────────────────────────────────────────────┘

  1. Abre el navegador en:  http://127.0.0.1:8000/
  
  2. Carga una imagen usando uno de estos métodos:
     • Arrastrar y soltar
     • Botón "Seleccionar Archivo"
     • Botón "Tomar Foto"
  
  3. Espera 2-5 segundos mientras se procesan las 15 etapas
  
  4. Explora los resultados en la pestaña "Preprocesamiento":
     • Desplázate hacia abajo
     • Verás la "Galería de 15 Etapas"
     • Cada tarjeta tiene imagen + estadísticas
  
  5. Navega por las otras pestañas:
     • 🖼️  Imágenes: Resultado final y paleta
     • 🔬 Análisis: Detección facial y colores
     • ✨ Recomendaciones: Sugerencias de outfit

┌──────────────────────────────────────────────────────────────────────────┐
│                        🔧 TECNOLOGÍAS UTILIZADAS                         │
└──────────────────────────────────────────────────────────────────────────┘

  Backend:
    • Django 5.2.7              Framework web
    • OpenCV 4.12.0             Procesamiento de imágenes
    • NumPy                     Operaciones matriciales
    • Matplotlib/Seaborn        Generación de gráficos
    • Pillow (PIL)              Manipulación de imágenes

  Frontend:
    • Bootstrap 5               Framework CSS
    • Chart.js                  Gráficos interactivos
    • Plotly.js                 Visualizaciones 3D
    • Font Awesome              Iconografía
    • CSS Animations            Efectos visuales
    • Vanilla JavaScript        Lógica de interfaz

  Algoritmos:
    • CLAHE                     Ecualización adaptativa
    • Bilateral Filter          Suavizado con preservación de bordes
    • Non-local Means           Reducción de ruido avanzada
    • Unsharp Masking           Realce de nitidez
    • Gamma Correction          Ajuste de luminosidad
    • Edge Detection            Análisis de contornos
    • Color Space Conversion    Manipulación HSV/RGB
